# Task 2 – Student Digital Portfolio

## Platforms Set Up

### 1. GitHub (github.com)
GitHub is a platform for hosting and sharing code using Git version control. It is widely used by developers, students, and researchers to collaborate on software projects and build a public coding portfolio. I created a profile README repository displaying my name, branch, current year, and a brief statement about my learning goals.

### 2. LinkedIn (linkedin.com)
LinkedIn is the world's largest professional networking platform. It is used to connect with peers, professors, recruiters, and industry professionals. I updated my Education section with: B.Tech in Computer Science & Engineering, VIT Bhopal University, Expected Graduation: 2028.

### 3. Kaggle (kaggle.com)
Kaggle is a platform for data science and machine learning competitions, datasets, and notebooks. It is used by students and professionals to practice data analysis, learn machine learning concepts, and showcase analytical skills to potential employers.

## Reflection
Setting up these three platforms gave me a clearer picture of the professional digital landscape. GitHub will be my most active platform once I start coding projects in my second semester — I plan to upload every lab assignment there. LinkedIn surprised me with how detailed a profile can be even at this early stage; I did not realise I could document skills and certifications alongside education. Kaggle is completely new to me, but I can see how useful it will be when I begin studying data structures and later machine learning in higher semesters. Over the next four years, I plan to use GitHub as a coding diary, LinkedIn for internship applications from third year onwards, and Kaggle to explore data-driven problem-solving.
